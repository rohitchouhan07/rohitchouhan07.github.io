Software I am currently using

Operating System

I use Arch Linux with DWM as the window manager on my personal machine. DWM is an excellent tiling window manager and changes the way that you interact with your computer. I highly recommend DWM to everyone on Linux. It is far more efficient than a traditional desktop environment, and also looks very cool. :P

Text editor and IDE
